# Папка для аудио интервью

Поместите сюда аудиофайлы интервью в формате MP3, OGG или WAV.

## Структура файлов:

- `interview-1.mp3` - Эхо древности
- `interview-2.mp3` - Механический вальс
- `interview-3.mp3` - Шёпот природы
- `interview-4.mp3` - Голоса города
- `interview-5.mp3` - Космическая тишина
- `interview-6.mp3` - Пульс времени

## Рекомендации:

- Формат: MP3 (AAC для лучшей совместимости)
- Битрейт: 128-192 kbps
- Длительность: 2-5 минут
- Моно или стерео

## Примечание:

Файлы будут автоматически подключены к соответствующим экспонатам в sound-exhibition.html
